# Songbook

This app lets users to save their favourite songs to sing & play

## Technology
- React 18.2.0
- Typescript 4.7.4
- Node 16.16


## Setup
- clone repository
- go to the root directory
- run `npm install` to install all required dependecies
- run `npm run start` to run the app locally
